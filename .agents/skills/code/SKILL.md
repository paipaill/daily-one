# Code Skill

这个技能用于进行代码编写、重构和性能优化。在执行此任务时，你需要：

## 1. 代码规范与最佳实践
- 遵循原生 HTML/CSS/JS 的最佳实践（针对当前零依赖的单文件架构项目）。
- 保持界面极简风格，使用语义化 HTML 标签。
- 确保 CSS 样式清晰，分离功能控制类（如 `.hidden`, `.active`）和展示层。
- JavaScript 代码需注意作用域，避免全局变量污染，尽可能使用现代 ES6+ 语法。

## 2. 需求实现流程
- **理解需求**：充分分析用户要求增加或修改的具体功能。
- **定位代码**：准确定位 `index.html` 或相关文件中的逻辑部分（如状态管理、API 请求、UI 渲染）。
- **编写代码**：实现功能时兼顾性能和可维护性。所有本地存储（`localStorage`）操作要做好异常捕获。
- **功能验证**：确保新添加的逻辑不会破坏原有核心业务，如每日数据拉取、缓存逻辑、UI状态流转（单向阅读、不可回退等）。

## 3. 性能与安全优化
- DOM 操作尽量批量执行，或使用碎片（DocumentFragment）及 `requestAnimationFrame` 优化性能。
- 敏感信息（如 API Key）必须有良好的安全和占位策略，并在提交前自动脱敏。
- 减少不必要的网络与 API 请求，充分利用并管理好 `localStorage` 缓存的生命周期。

## 4. 输出规范
- 提交修改前，确认代码整洁、排版清晰。
- 针对复杂的业务或视觉重构变更，必须添加中文注释详细阐明修改原因或交互机制。
